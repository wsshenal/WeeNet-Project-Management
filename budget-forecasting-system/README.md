# Budget Forecasting System - Standalone Project

## Project Structure
```
budget-forecasting-system/
├── backend/           # Flask API
├── frontend/          # React + Vite
├── data/             # Training data
└── README.md         # This file
```

## Tech Stack
- **Backend:** Flask, XGBoost, scikit-learn, OpenAI
- **Frontend:** React, Vite, Ant Design, Recharts
- **ML:** XGBoost Regression

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- ML-based budget prediction
- AI-powered insights
- Beautiful UI with charts
- Export to PDF
- Real-time predictions
